# Bank Customer Churn - Feature Importance Analysis

## Overview
This project analyzes customer churn behavior and identifies the most influential features affecting customer attrition.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Objectives
- Build a churn prediction model
- Analyze feature importance
- Interpret model results

## Dataset
Bank Customer Churn Dataset
